# Exisiting repository checker

This python code cautomatically checks if repository exists.

## USAGE

```bash
check_repo.py <repo_config_file>

```

config file format:

```bash
https://github.com/<username1>/<repo_name1>
https://github.com/<username2>/<repo_name2>
...

```
